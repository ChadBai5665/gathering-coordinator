# Ontheway 视觉设计需求文档

> 用于 UI 设计软件（Figma/Sketch/AI 设计工具）的完整设计指引

---

## 一、品牌信息

**项目名称**：碰个头 (Ontheway)
**定位**：社交聚餐应用，帮助朋友们快速约饭、选餐厅、分摊费用
**核心理念**：轻松、温暖、社交、美食

**品牌色彩**：
- 主色调：橙色 (#FF6B35 / rgb(255, 107, 53))
- 辅助色：青色 (#14B8A6 / Teal-500)
- 渐变：从橙色到青色的温暖渐变

**设计风格**：
- 现代、圆润、友好
- 使用圆角设计
- 温暖的色彩搭配
- 简洁不复杂

---

## 二、Logo 设计需求

### 主 Logo（应用图标）

**文件命名**：`logo-main.png`

**设计要求**：
- 尺寸：1024x1024px（高清原图，后续自动生成其他尺寸）
- 形状：圆角方形（圆角半径约 22%）
- 背景：橙色到青色渐变（左上到右下）
- 图标元素：
  - 主题：餐饮 + 社交的结合
  - 建议元素：餐具、对话气泡、位置标记、聚会的人群等
  - 风格：简洁图形，白色或浅色线条
  - 避免过于复杂的细节
- 可识别性：缩小到 40x40px 时仍清晰可辨

**用途**：
- 小程序图标（144x144px）
- Web 登录页主 Logo（64x64px）
- 小程序登录页 Logo（120x120px）
- Web Header 小图标（40x40px）

---

### Favicon（浏览器图标）

**文件命名**：`favicon.png`

**设计要求**：
- 尺寸：512x512px（原图，后续自动生成 32x32、16x16）
- 形状：方形或圆形
- 设计：主 Logo 的简化版本，只保留核心图形
- 背景：可以是纯色或渐变
- 要求：在 16x16px 时仍能识别

---

## 三、TabBar 图标设计需求

> 小程序底部导航栏的三个图标，每个图标需要「未选中」和「选中」两个状态

### 3.1 首页图标

**文件命名**：
- 未选中：`tabbar-home.png`
- 选中：`tabbar-home-active.png`

**设计要求**：
- 尺寸：162x162px（2倍图，实际显示 81x81px）
- 图标主题：首页 / 发现 / 动态
- 建议元素：房子、首页、动态流、发现等
- 未选中状态：灰色 (#9CA3AF / Gray-400)
- 选中状态：橙色 (#FF6B35)
- 线条粗细：中等（约 3-4px）
- 风格：圆润、现代、线性图标

---

### 3.2 聚会列表图标

**文件命名**：
- 未选中：`tabbar-list.png`
- 选中：`tabbar-list-active.png`

**设计要求**：
- 尺寸：162x162px（2倍图）
- 图标主题：聚会列表 / 活动 / 日历
- 建议元素：列表、日历、多人聚会、活动卡片等
- 未选中状态：灰色 (#9CA3AF)
- 选中状态：橙色 (#FF6B35)
- 线条粗细：中等（约 3-4px）
- 风格：与首页图标保持一致

---

### 3.3 个人中心图标

**文件命名**：
- 未选中：`tabbar-user.png`
- 选中：`tabbar-user-active.png`

**设计要求**：
- 尺寸：162x162px（2倍图）
- 图标主题：个人中心 / 我的 / 用户
- 建议元素：用户头像、个人、设置等
- 未选中状态：灰色 (#9CA3AF)
- 选中状态：橙色 (#FF6B35)
- 线条粗细：中等（约 3-4px）
- 风格：与其他图标保持一致

---

## 四、地图标记图标

### 4.1 用户位置标记

**文件命名**：`marker-user.png`

**设计要求**：
- 尺寸：96x96px（2倍图，实际显示 48x48px）
- 图标主题：用户当前位置
- 建议元素：人物图标 + 位置标记的结合
- 颜色：橙色 (#FF6B35) 或渐变
- 背景：白色圆形底，带阴影
- 风格：清晰、易识别

**用途**：在地图上标记参与聚会的用户位置

---

### 4.2 餐厅位置标记

**文件命名**：`marker-restaurant.png`

**设计要求**：
- 尺寸：96x96px（2倍图）
- 图标主题：餐厅位置
- 建议元素：餐具图标 + 位置标记的结合
- 颜色：青色 (#14B8A6) 或与用户标记区分的颜色
- 背景：白色圆形底，带阴影
- 风格：与用户标记保持一致，但颜色区分

**用途**：在地图上标记候选餐厅位置

## 五、设计输出规范

### 5.1 文件格式
- 所有图标：PNG 格式，透明背景（除非特别说明需要背景）
- 颜色模式：RGB
- 分辨率：72 DPI（屏幕显示）

### 5.2 命名规范
严格按照上述文件命名，不要添加额外后缀或前缀

### 5.3 输出清单

请按以下顺序输出文件：

```

5. tabbar-list.png        (162x162px, 列表-未选中)
```

## 六、设计参考

### 风格参考
- 类似应用：美团、大众点评、饿了么（但更温暖、更社交化）
- 图标风格：iOS 风格的线性图标，圆润友好
- 色彩参考：温暖的橙色系 + 清新的青色系

### 避免的设计
- ❌ 过于复杂的细节（缩小后看不清）
- ❌ 过细的线条（在小尺寸下会模糊）
- ❌ 过于严肃或商务的风格
- ❌ 使用过多颜色（保持简洁）

## 七、验收标准

### Logo 验收
- [ ] 在 40x40px 缩小时仍清晰可辨
- [ ] 渐变色彩自然过渡
- [ ] 图标元素与品牌定位相符
- [ ] 圆角比例协调

### TabBar 图标验收
- [ ] 三个图标风格统一
- [ ] 线条粗细一致
- [ ] 选中/未选中状态对比明显
- [ ] 在 81x81px 显示时清晰

### 地图标记验收
- [ ] 用户标记和餐厅标记易于区分
- [ ] 在地图上显示清晰
- [ ] 带阴影效果，有立体感
